Find and add YouTube videos to a week's resources in the Model UN Club website.

Arguments format: `week number - topic or specific request`
Example: `1 - add a video about the Nile dam dispute`

$ARGUMENTS

Steps:
1. Find the week card in `index.html`
2. Add the requested YouTube video(s) as embedded iframes in the video-grid
3. Use the same format: `<div class="video-card"><iframe src="https://www.youtube.com/embed/VIDEO_ID" allowfullscreen loading="lazy"></iframe><div class="video-title">Title — Channel</div></div>`
4. Git commit and push
5. Confirm what was added
