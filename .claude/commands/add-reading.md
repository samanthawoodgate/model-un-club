Add further reading links to a week's resources in the Model UN Club website.

Arguments format: `week number - topic or specific URLs`
Example: `3 - add some resources about OPEC from academic sources`

$ARGUMENTS

Steps:
1. Find the week card in `index.html`
2. Add links to the reading-list section using the format: `<a href="URL" target="_blank">Title<span class="source">domain.com</span></a>`
3. Prefer reputable sources: UN agencies, BBC, Reuters, academic institutions, established NGOs
4. Git commit and push
5. List what was added
